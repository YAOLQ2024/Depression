**第一部分：项目技术资产总览**
主要技术栈：
- Web 应用：Flask + Jinja2 + SQLite + 原生 JS/Alpine.js + Tailwind CDN + Chart.js + marked.js
- 模型接入：OpenAI 兼容聊天接口封装、本地 vLLM、FastAPI RAG 服务
- 语音：浏览器 `SpeechRecognition`、浏览器 `speechSynthesis`、后端 Dolphin ASR
- 视觉：ONNX Runtime GPU 人脸检测/表情识别
- 脑电：串口 EEG 接收与规则分类
- 模型工程：Transformers / LangChain / FAISS / xtuner / 数据集处理脚本 / 多个本地模型目录
